Milestone Project — Phase 1
End-to-End Exploratory Data Analysis Report
Choose a public dataset (e.g. from Kaggle or UCI ML Repository). Write a complete EDA notebook: load and clean the data, visualise distributions, compute correlations, test a hypothesis with a statistical test, and write a 500-word summary of your findings. This demonstrates that your math, Python, and data instincts all work together.
Jupyter Notebook 5+ Visualisations Hypothesis Test Written Summary GitHub Repo

Milestone Project — Phase 2
Business Prediction Model with Full Report
Build a real business prediction model — churn prediction, credit scoring, or demand forecasting. Compare at least 3 algorithms, tune hyperparameters, evaluate rigorously, and produce a non-technical business brief (1–2 pages) alongside the technical notebook. This is the core of consulting work: translating ML into business value.

Milestone Project — Phase 3
Fine-Tuned NLP Application
Fine-tune a pre-trained transformer (e.g., BERT or DistilBERT via HuggingFace) on a domain-specific text classification or summarisation task relevant to a business use case (e.g., customer support ticket routing, review sentiment). Deploy it as a simple API with FastAPI or Gradio. Document the performance, limitations, and estimated business value.
